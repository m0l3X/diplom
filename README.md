# ass rpg game
It's our cool project rpg-novel game made as a graduation project for our college.

<line>
To use AI in battles you must set up your own llama-cpp server on your pc and run it on port 8080. Also, set it's ip address in player_database savefile, etc. 0.0.0.0 if you're running it on your own machine, or 192.168.1.something if it's discrete PC on your LAN.

it uses bare requests to v1/completions endpoint

also ofc you can put domains in there, so you can point to your own vds server.
